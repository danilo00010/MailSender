## Mail Sender

This is a script for hosting an API that handles a few data to send the same email using any SMTP Service to a list of destination emails.

#### Changelog:

```
- Bun
- Typescript
- ElysiaJS
- .env
```

### Installation

```
bun install
```

You must configure the <code>.env</code> file.

### Usage

```
bun run dev
```

### Test

```
bun test
```
